# instacompute.github.io

Disclaimer: For educational purposes only. Not financial advice. Consult a professional advisor.

Calculators:

* [Bucket-based FI-ready Planning Calculator](/bucket-calc.html)
    * Configurable Return rates (default: conservative, post-tax returns default)
    * Simulation models: deterministic (fixed returns based), monte carlo (single & batch), nifty50/historical
    * Rebalancing assumption:
        * Only from Equity to Debt
        * Debt to PPF during build-up phase, PPF to Debt switch during old age
        * Rebalanced amount maximum of equity allocation threshold (excess) and deficit against number of expenses in debt bucket (default: 15 years expenses in debt bucket)
